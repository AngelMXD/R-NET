# 2.0 ❯ Glosario

Por
Cooper
●
Publicado 2025-03-11

Bienvenido al glosario de términos usados en Discord. Este documento está organizado por categorías para que encuentres fácilmente lo que buscas. Usa la tabla de contenidos a continuación para navegar entre secciones.

# 🏗️ Conceptos básicos

  • Servidor: Espacio principal (comunidad, club, grupo de amigos). Es la organización raíz donde se gestionan miembros, canales y configuraciones.
  • Canal: Sección interna (texto o voz). Cada canal tiene su propio historial de mensajes y ajustes de permisos.
  • Categoría: Agrupa canales. Sirve para ordenar y ocultar canales según el tema o el nivel de acceso.
  • Roles: Etiquetas con permisos (Moderador, VIP, etc.). Los roles controlan lo que los usuarios pueden ver y hacer en el servidor.
  • Emoji / Emote: Pequeña imagen que se inserta en mensajes. Puede ser global o de servidor; también existen emojis animados para usuarios Nitro.
  • Invitación: Enlace o código que permite a otros unirse a un servidor. Los administradores deciden si caduca o tiene un límite de usos.
  • Servidor público/privado: Define si cualquiera puede buscarlo o si necesita invitación. Los públicos aparecen en los resultados de búsqueda de Discord.

# 💬 Comunicación y funciones

  • DM / MD: Mensaje directo privado. Se usa para charlas uno‑a‑uno o pequeños grupos fuera de los canales.
  • Mencionar (@): Notificar a alguien usando @nombre. Genera alerta y, a veces, sonido si la persona tiene notificaciones activas.
  • Ping: Acto de mencionar (sinónimo de “notifiqué”). A veces se usa en mensajes para llamar la atención de un grupo.
  • Ghost ping: Mención borrada inmediatamente. Considerada una forma de molestar porque deja notificación sin contenido visible.
  • Reacción: Emoji añadido debajo de un mensaje. Sirve para reaccionar sin escribir y contabiliza votos o estados de ánimo.
  • Sticker: Imagen estática o animada que se envía como mensaje. Se accede desde una biblioteca separada y puede requerir Nitro.
  • Thread (Hilo): Subconversación enlazada a un mensaje de canal. Ideal para debates que no quieres que llenen el chat principal.
  • Pinned (Fijado): Mensaje guardado en la lista de destacados del canal. Facilita volver a mensajes importantes o reglas.

# 🛡️ Moderación y gestión

  • Mod: Moderador. Usuario con permisos especiales para gestionar comportamientos y contenido.
  • Ban: Expulsión permanente. El usuario no puede reentrar a menos que se le levante el baneo.
  • Kick: Expulsión temporal; puede volver con invitación. No elimina historial, solo desconecta al jugador.
  • Timeout: Silencio temporario para “enfriar”. El usuario sigue en el servidor pero no puede hablar ni reaccionar.
  • Bot: Programa automático (MEE6, Dyno, etc.). Puede moderar, reproducir música, enviar estadísticas o responder comandos.
  • Warn / Advertencia: Aviso formal a un usuario por incumplir normas. Muchas comunidades lo documentan en el canal de staff.
  • Audit log: Registro interno de acciones de moderación y configuraciones. Muestra quién cambió qué y cuándo.
  • Nickname (Apodo): Nombre personalizado que un usuario puede tener por servidor. Ayuda a identificar roles u horarios.
  • Verificación: Ajuste que exige correo/edad u otros requisitos antes de participar. Reduce bots y trolls.

# 👾 Personajes y cultura oficial de Discord

  • Wumpus: Mascota oficial. Aparece en mensajes de error y material promocional de Discord.
  • Clyde: Bot de sistema. Envía notificaciones automáticas cuando se cometen errores o se ejecutan comandos inválidos.
  • Nitro: Suscripción premium. Ofrece emojis animados, mayor límite de subida de archivos y más.
  • Server Boost: Mejora desbloqueable por Nitro. Sube el nivel del servidor y desbloquea funciones globales.
  • HypeSquad: Programa de “casas” con insignia. Los usuarios reciben noticias exclusivas y representaciones en eventos.
  • Easter eggs: Mensajes o imágenes ocultas espaladas por la interfaz. Normalmente descubiertas por exploradores o celebraciones.

# 🗣️ Jerga general

  • LFG (Looking For Group): Buscando grupo. Normalmente se publica en canales específicos para juegos o eventos.
  • AFK (Away From Keyboard): Ausente del teclado. Algunos bots marcan automáticamente a usuarios inactivos.
  • Lurker: Usuario que observa sin participar. No genera notificaciones y suele leer solo.
  • Ratio / Ratioed: Mensaje con muchas reacciones negativas. «Le dieron ratio» indica rechazo colectivo.
  • Copypasta: Texto que se copia y pega repetidamente. A menudo contiene bromas internas o memes de larga duración.
  • OOMF: Uno de mis amigos/seguidores. Se usa cuando la identidad del contacto no importa o para preservar privacidad.
  • RP (Roleplay): Actividad de interpretar personajes. Común en servidores literarios o de juegos de mesa.
  • Pinged: Estado de haber recibido una mención. Puede activar notificación o sonido dependiendo de configuración.
  • Clout: Reputación o prestigio digital; usado en memes y bromas. «Buscar clout» significa actuar para llamar la atención.

# 🛡️ Jerga de moderación y seguridad

  • TOS: Términos de servicio. Documento legal de Discord que los usuarios aceptan al registrarse.
  • Raid: Ataque de spam coordinado. A veces impulsado por invitados maliciosos o bots automatizados.
  • Nuke / Nukear:
    1. Borrar y recrear canal para limpiar spam.
    2. Ataque que borra todo el servidor.
  • Alt: Cuenta alternativa usada para evadir baneo. Los «altosa» pueden complicar el seguimiento de infracciones.
  • Modmail: Bot que envía DMs a un canal solo para mods. Facilita comunicación privada entre usuarios y moderadores.
  • RAID (como sigla de "Redundant Array of Independent Disks"): Broma técnica cuando se habla de backups de servidores. No tiene nada que ver con ataques, es un juego de palabras.

# 🏷️ Etiquetas y estados

  • SFW / NSFW: Contenido seguro o no seguro para el trabajo. Los canales NSFW requieren edad y advierten antes de entrar.
  • Spoiler: Texto o imagen ocultos con ||spoiler||. Evita arruinar tramas o sorpresas para otros.
  • Muteado / Ensordecido:
    • Muteado: Micrófono apagado, no se te escucha.
    • Ensordecido: No escuchas a nadie, aunque ellos sí te oyen.
  • Invisible: Estado que hace que parezcas desconectado. Permite usar Discord sin alertar a otros.
  • Do Not Disturb (DND): No recibir notificaciones. Muestra un icono rojo y bloquea alertas de mensajes.
  • Online / Idle / Offline: Estados de presencia. Idle aparece cuando el usuario está inactivo por un tiempo.

# 🚫 Hábitos y comportamientos a evitar

  • Spam: Mensajes repetitivos/no solicitados. Suelen perjudicar la legibilidad del chat.
  • Flood: Mensajes cortos en rápida sucesión. Empuja la conversación hacia arriba y molesta a lectores.
  • Walltext: Texto muy largo sin formato. Difícil de leer y responde mal en móviles.
  • Ping spam / Mass ping: Etiquetas abusivas (@everyone/@here). Causan notificaciones masivas y enfadan a usuarios.
  • Bait: Mensaje diseñado para provocar. Busca reacciones emocionales o peleas.
  • Shitposting: Contenido absurdo/meme sin contexto. A veces aceptado en canales específicos pero mal visto en canales serios.
  • Scripting: Uso de bots o macros para automatizar mensajes abusivos. Viola TOS y puede resultar en baneo.
  • Snipe / Snipar: Recuperar mensajes borrados con bots. Usado por moderadores o trolls para levantar drama.
  • Alt farming: Crear múltiples cuentas para manipular votos o eventos. También se le llama «sockpuppet».
